# 3 Cards Game

A simple, premium-style card game built with HTML, CSS, and JavaScript.

## How to Play
- **Goal**: Keep your points low!
- **Values**: J, Q, K = 10 pts; A = 1 pt; Others = Face Value.
- **Turn**: Discard 1 card, then Draw 1 card.
- **Show**: If you think you have lower points than the computer, click "Show".
- **Winning**: Lowest points wins the round (0 pts). Loser gets hand value added to their score.
- **Game Over**: When someone hits 100 pts.

## Deployment

### Option 1: Netlify Drop (Easiest)
1. Locate the `3-cards-game.zip` file in this directory.
2. Go to [Netlify Drop](https://app.netlify.com/drop).
3. Drag and drop the zip file (or the folder) onto the page.
4. Your game will be online instantly!

### Option 2: GitHub Pages
1. Create a new repository on GitHub.
2. Push the contents of this folder to the repository.
3. Go to Settings > Pages and enable GitHub Pages for the `main` branch.

### Option 3: Surge.sh (CLI)
1. Run `npx surge` in this directory.
2. Follow the prompts to create an account and deploy.
